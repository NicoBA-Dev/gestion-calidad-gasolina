# ⛽ Sistema de Gestión de Calidad de Gasolina (FuelQuality MS)

Sistema web especializado para el monitoreo de laboratorio, control de parámetros de calidad (octanaje, densidad, azufre), trazabilidad de cisternas y emisión de certificados de conformidad para combustibles.

## 📋 Módulos Principales

1. **Recepción & Trazabilidad de Cisternas:** Registro de lotes de entrada, precintos de seguridad y proveedor.
2. **Análisis de Laboratorio:** Módulo para la captura de pruebas de octanaje (RON/MON), densidad a 15°C, presión de vapor y azufre.
3. **Control de Calidad & Alertas:** Detección automática cuando un lote no cumple con los estándares normativos.
4. **Certificación Digital:** Emisión de certificados de calidad de combustible aprobados para distribución.
